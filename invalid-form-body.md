# Getting an invalid form body error?

 - Are you on mobile?
   - If so, check for updates in your app store of choice, and make sure you have the latest version.
     - IF that doesn't fix it, try reinstalling your app.
 - Are you on Desktop/Web?
   - If so, try refreshing Discord with CTRL/CMD + R.
 - Are you on a mobile browser?
   - If so, that's unsupported by both Discord and me.
 - If the above didn't fix it and you're not using the browser app on mobile, please file a bug report.
